Javascript Best Practices
1. Kies tussen de Object literal of Object constructor: kijk of je iets eenmalig wil aanmaken of meerdere malen. 
2. Maak een self invoking anonymous function: Alles wat binnen die functie bestaat is afgeschermd van wat er buiten gebeurt.
3. Gebruik namespace: een globale variabele
4. Verdeel je code logischerwijs over meerdere .js bestanden, zoals utils.js. 
5. Maak namen in je code door middel van camelCasing: elk los woord begint met een hoofdletter.
